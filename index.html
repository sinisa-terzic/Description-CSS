Layout

Container:

☛ The .container class is used to define the styles for the container element.

☛ width and max-width are set to control the width of the container. max-width ensures that the container won't exceed a certain width, especially on larger screens.

html

  • content, content, content
  • content, content, content
  • content, content, content

css

  • /* 120px * 10px = 1200px */
  • .container {
  • width: 80%;
  • max-width: 120em;
  • margin: 0 auto;
  • }

Grid:

☛ The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

☛ Class for grid: .grid

html

css

  • .grid {
  • display: grid;
  • }

Flex:

☛ The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

☛ Class for flex: .flex

html

css

  • .flex {
  • display: flex;
  • }

Hyperlinks:

☛ The a tag defines a hyperlink, which is used to link from one page to another. Class for hyperlink .link

☛ The most important attribute of the a element is the href attribute, which indicates the link's destination. We also have optional attribute target

html

  • Link

css

  • .link:link,
  • .link:visited {
  • display: inline-block;
  • color: #e67e22;
  • text-decoration: none;
  • transition: all 0.3s;
  • border-bottom: 2px solid transparent;
  • }
  • .link:hover,
  • .link:active {
  • color: #ff9f4b;
  • border-bottom: 2px solid currentColor;
  • }
